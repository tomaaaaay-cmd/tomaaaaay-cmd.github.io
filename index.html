Sync between your phones
Because this runs in a browser, the easiest free sync option is JSONBin.io.

Steps:
1. Go to jsonbin.io and create a free account
2. Create a new "bin" — copy the Bin ID
3. Copy your API key from the dashboard
4. Enter both below and tap Save

Both phones will share the same cloud bin. Each user's data stays separate inside it.
