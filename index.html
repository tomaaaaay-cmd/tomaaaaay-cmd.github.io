Sync across devices
To sync between Tommy's Android and Wife's iPhone, save this app's data using a free cloud storage link. Tap the button below to get setup instructions.
Add LiftLog to your home screen
On iPhone (Safari):
1. Open this page in Safari
2. Tap the Share button (box with arrow)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add" — done!

On Android (Chrome):
1. Open this page in Chrome
2. Tap the three-dot menu (top right)
3. Tap "Add to Home screen"
4. Tap "Add" — done!
Sync between your phones
Because this runs in a browser, the easiest free sync option is JSONBin.io — a free cloud database.

Steps:
1. Go to jsonbin.io and create a free account
2. Create a new "bin" — copy the Bin ID
3. Copy your API key from the dashboard
4. Come back here and tap "Enter sync keys" below

Once set up, both phones will read and write to the same cloud bin. Each user's data stays separate inside it.
